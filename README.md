# Moderat
Fun Made Windows Remote Administration Tool in Python + GUI

Requirements
- Windows (x86, x64)
- Python 2.7
- PyQt4
- PIL

Features
- Execute Shell Commands Remotely
- Remote File Explorer
- Desktop Preview
- Password Protection
- To Be Continued...
